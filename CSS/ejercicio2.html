Una página web que tenga en la cabecera la siguiente definición de estilos para las etiquetas párrafo: borde verde, color de fondo negro y texto en blanco.

Escribir dos párrafos con un pequeño texto dentro. En el segundo se aplicará un estilo en la propia etiqueta poniendo el color del texto rojo. ¿De qué color sale el texto de cada párrafo? ¿Por qué?

El texto del primer párrafo es blanco, y el del segundo es rojo, porque tiene prioridad el estilo que está en la propia etiqueta
